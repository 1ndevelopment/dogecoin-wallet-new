Dogecoin Wallet

Dogecoin Wallet app for your Android device. Standalone Bitcoin node, no centralized backend required. Forked from the popular Bitcoin Wallet.

View project onGitHub

Welcome to Dogecoin Wallet.

Dogecoin Wallet for Android lets you keep Dogecoins right with you on your phone. It is a port of the Bitcoin Wallet for Android by @schildbach.

About

The app is based on DogecoinJ, which in turn is a port of BitCoinJ (@bitcoinj). You can find DogecoinJ here: https://github.com/langerhans/dogecoinj-new I ported all the needed changes for Dogecoin compatibility over.

Download

You can get the app directly from the Google Play store: https://play.google.com/store/apps/details?id=de.langerhans.wallet

Alternatively you can download the current version for sideloading right here on this page.

Contributing

Building

Basicially you clone both this repo and DogecoinJ and compile it with Maven. More info is included in the README file.
